# Instrucciones para agregar tu CV

Para que el botón "Descargar CV" funcione, necesitas:

1. **Crear tu CV en formato PDF**
   - Usa Canva, Google Docs, Word, etc.
   - Guárdalo como PDF

2. **Nombrar el archivo**
   - Nombre: `Matteo-CV.pdf`

3. **Colocarlo aquí**
   - Ruta: `assets/cv/Matteo-CV.pdf`

4. **Reemplazar este archivo README**
   - Elimina este archivo README.md
   - Coloca tu PDF aquí

## Plantillas recomendadas:

- Canva: https://www.canva.com/templates/s/resume/
- FlowCV: https://flowcv.com/
- Resume.io: https://resume.io/

El botón ya está configurado para descargar automáticamente el archivo cuando lo coloques aquí.
